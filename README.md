Calculadora de Operaciones Matemáticas en PHP

Este proyecto consiste en una calculadora básica desarrollada en PHP, utilizando HTML y CSS, que permite realizar operaciones matemáticas simples y demostrar el uso del orden de precedencia.

Funcionalidades

El programa permite realizar las siguientes operaciones:

Suma
Resta
Multiplicación
División
Orden de precedencia

La calculadora respeta las reglas matemáticas estándar:

Paréntesis
Multiplicación y división
Suma y resta
